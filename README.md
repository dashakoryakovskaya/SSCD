# Модели и методы полуконтролируемого кросс-доменного обучения текстовых представлений с адаптивным согласованием латентных признаков в архитектурах больших языковых моделей
ВКР НИУ ВШЭ Коряковская Дарья

---
**Цель работы**  заключается в разработке и экспериментальной проверке новых нейросетевых моделей и методов полуконтролируемого **кросс-доменного** обучения текстовых представлений, основанных на адаптивном согласовании **латентных признаков в больших языковых моделях**, в контексте эмпатичного искусственного интеллекта, направленных на повышение качества обобщения и переноса знаний между доменами при ограниченных аннотированных данных. Особое внимание уделяется проверке эффективности предложенных методов на задачах распознавания эмоционального состояния и личностных качеств человека, что позволяет оценивать их потенциал для построения адаптивных, контекстно-чувствительных интеллектуальных систем.
